% podman-secret-rm(1)

## NAME
podman\-secret\-rm - Remove one or more secrets

## SYNOPSIS
**podman volume rm** [*options*] *secret* [...]

## DESCRIPTION

Removes one or more secrets

## OPTIONS

#### **--all, -a**
Remove all existing secrets

Print usage statement
#### **--help**

Print usage statement


## EXAMPLES

```
$ podman secret rm mysecret1 mysecret2
```
